# ⬡ OPPerpDEX

**The first decentralized perpetual futures exchange on OPNet Bitcoin L2.**

Trade BTC, ETH, SOL and more with up to 50× leverage. Non-custodial, sub-cent fees, powered by Bitcoin.

---

## 🚀 Live Demo

Deployed on OPNet Testnet — [View Live](https://op-perpdex.vercel.app)

---

## 🔧 Getting Started (Testnet)

1. **Install OP_WALLET** → [Chrome Web Store](https://chromewebstore.google.com/detail/opwallet/pmbjpcmaaladnfpacpmhmnfmpklgbdjb)
2. **Get free tBTC** → [faucet.opnet.org](https://faucet.opnet.org)
3. **Connect & Trade** → Click "Connect Wallet" and select OP_WALLET

---

## 🏗 Tech Stack

- **Frontend**: React 18 + Vite
- **Blockchain**: OPNet Bitcoin L2 (Testnet 3)
- **Wallet**: OP_WALLET (+ UniSat, OKX, Xverse)
- **Contracts**: 4 deployed smart contracts (AssemblyScript → WASM)

---

## 📦 Local Development

```bash
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173)

## 🚢 Deploy to Vercel
